Как запустить:
1. Скачать Python
2. Скачать репозиторий
2.1. Сконфигурировать версию Python в приложениях типа PyCharm
2.2. Установить requirments(в PyCharm через интерфейс или в консоли через pip install -r requirements.txt)
3. Открыть терминал в папке проекта
4. Написать комманду для запуска сервера uvicorn app:app --reload
5. Открыть в браузере: http://localhost:8000
