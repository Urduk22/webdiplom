Как запустить:
1. Скачать Python
2. Скачать репозиторий
3. Сконфигурировать версию Python в приложениях типа PyCharm
4. Открыть терминал в папке проекта
5. написать py -m venv venv для установки виртуального окружения
6. написать .\venv\Scripts\Activate.ps1 для запуска виртуального окружения
6. Установить requirments(в PyCharm через интерфейс или в консоли через pip install -r requirements.txt)
7. Написать комманду для запуска сервера uvicorn app:app --reload
8. Открыть в браузере: http://localhost:8000
